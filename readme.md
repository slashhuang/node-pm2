# node-pm2

> pm2 is the most commonly used deploy tool for nodejs project

> this repo is a learning project for pm2

## command

```bash

    npm install

    # use nodemon to start a simple http server
    npm run dev

    # start app with pm2
    npm run pm2

    # start app with pm2 in cluster mode
    npm run cluster
```
